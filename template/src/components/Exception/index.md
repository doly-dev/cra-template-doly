# Exception - 异常

> 常见的有 `304` `404` `500`

异常页用于对页面特定的异常状态进行反馈。通常，它包含对错误状态的阐述，并向用户提供建议或操作，避免用户感到迷失和困惑。

## API

| 参数    | 说明     | 类型          | 默认值 |
| ------- | -------- | ------------- | ------ |
| icon    | 图片     | `ReactNode`   | -      |
| title   | 标题     | `ReactNode`   | -      |
| desc    | 补充描述 | `ReactNode`   | -      |
| extra   | 额外信息 | `ReactNode`   | -      |
| actions | 操作     | `ReactNode[]` | -      |
