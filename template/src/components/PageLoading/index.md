# PageLoading - 加载中、加载失败

[`asyncComponent`](/docs/component-async) 异步组件加载的提示页面。包含加载中、加载失败。

## API

默认展示加载中页面。当 `error` 为 `true` 时，显示加载失败页面。

| 参数    | 说明               | 类型      | 默认值  |
| ------- | ------------------ | --------- | ------- |
| loading | 显示加载中         | `boolean` | `true`  |
| error   | 显示加载失败或报错 | `boolean` | `false` |
